# MedTest Lab Platform

## Tech Stacks (LAMP Stack)

- Container
  - Docker
  
- Frontend
  - HTML + CSS + JS

- Backend
  - Python 3

- Database
  - MySQL

## Code Usage

- To run the Docker file 

  ```bash
  docker-compose up -d --build
  ```

## Role Actions
- Lab Staff
  - Check & Add Test(s)
  - Check & Write Test Result(s)
- Patient
  - Check & Booking Appointment(s)
  - Check Test Result(s)
  - Pay Bill(s)
- Sectary
	- Manage Appointment(s)
	- Make order(s)
	- Confirm Payment
